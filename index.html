コーディング

Webデザインを実装する過程です。サイトの公開までしっかりコミットさせて頂きます。

デザイン

デザインを決める過程です。お客様のご希望にそって適切なデザインを提案します。

＊デザイン+コーディングももちろん承っております。簡易なサイトからでもOKです。
ご気軽にご連絡、相談など受け付けております。

スキル

  • HTML

    CSSとともに得意としているコーディングの分野です。

  • CSS

    HTMLとともに得意としている分野です。優しい印象からクールな印象まで幅広く対応しております。

  • JavaScript

    JavaScriptで動きをつけるサイトを構築でき、より幅広いアイデアを提案できます。

  • ワーキングスパン

    大学の授業は少なめですが多少忙しい日もあります。ゆえに納品期間の長短の差はあります。

  • その他

    全中国語や全英語などの、日本国外に向けたWEB制作も可能です。
    また、Photoshopやillustrater、ruby on railsなどさまざまなカテゴリーも網羅しております。ご気軽にご相談ください。

＊最大の特徴はコンタクトの緻密さとレスポンスの素早さにあります。
お客様のご満足・売上拡大を第一として、誠実にコミットさせていただきます。
